Roadmap

The next version of Haizea will be TP1.1, due mid-July 2008. This version will include the following improvements:

  • More complete integration with OpenNebula.
  • Documentation on how to set up and use OpenNebula with Haizea as a scheduling backend.
  • Tools to produce reports and pretty graphs based on Haizea runs (both simulation and OpenNebula)

There are several improvements that are planned for upcoming versions (coming out in summer and fall of 2008, exact dates to be determined):

  • Support for more types of leases
  • Decoupling policy decisions from Haizea
  • Lease negotiation
  • Command Line Interface (CLI) to Haizea
  • Processing asynchronous events from enactment backend

Wish list

  • Slot table data structure
  • General code cleanup
