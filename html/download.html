Downloads

Haizea

The current version of Haizea is Technology Preview 1.0, released on July 7th, 2008. To install, download haizea-TP1.0.tar.gz and follow the installation instructions. Haizea is published under the Apache License 2.0, a BSD-like OSI-compatible license.

Additional tracefiles

Additional tracefiles, including those used for the experiments in the paper Combining Batch Execution and Leasing Using Virtual Machines, will be posted shortly.

SVN repository

Our SVN repository can be found at http://haizea.cs.uchicago.edu/svn/. You can checkout the latest version of the code like this:

svn co http://haizea.cs.uchicago.edu/svn/trunk

You can also point your web browser to http://haizea.cs.uchicago.edu/svn/ to peruse the code. The current interface is very basic, but a pretty web-based SVN browser will be available further down the road.

To keep track of commmits, you can subscribe to the haizea-commit mailing list to get an e-mail notification every time there is a commit in the SVN repository.
