Downloads

Haizea

The current version of Haizea is Technology Preview 1.0, released on July 7th, 2008. To install, download haizea-TP1.0.tar.gz and follow the installation instructions. Haizea is published under the Apache License 2.0, a BSD-derivative OSI-compatible license.

Additional tracefiles

Additional tracefiles, including those used for the experiments in the paper Combining Batch Execution and Leasing Using Virtual Machines, will be posted shortly.

SVN repository

We are currently working on setting up a public SVN repository for the Haizea code. In the meanwhile, the downloadable tarball includes most of the source code.
