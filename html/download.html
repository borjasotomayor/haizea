Haizea

The current version of Haizea is Technology Preview 1.3, released on February 16th, 2009. To install, download haizea-TP1.3.tar.gz and follow the installation instructions. Haizea is published under the Apache License 2.0, a BSD-like OSI-compatible license.

SVN repository

You can checkout the latest version of the code like this:

svn co http://phoenixforge.cs.uchicago.edu/svn/haizea/trunk

You can also browse our SVN repository at PhoenixForge.

To keep track of commmits, you can subscribe to the haizea-commit mailing list to get an e-mail notification every time there is a commit in the SVN repository.

Additional tracefiles

Additional tracefiles, including those used for the experiments in the paper Combining Batch Execution and Leasing Using Virtual Machines can be found in the SVN repository (under trunk/traces/). They are not included in the distributable version of Haizea because of their size. If you have any questions about using these tracefiles, please don't hesitate to contact us.
